How to Use:

  • Load an image using the "Load Image" button
  • Left-click on an object to add foreground points (green)
  • Right-click to add background points (red) for refinement
  • The segmentation updates automatically after each click
  • Use "Reset Points" to clear points and try again
  • Use "Clear Image" to load a new image
